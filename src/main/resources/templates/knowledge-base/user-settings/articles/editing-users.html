Editing Users

To edit a user, select the button of the user you would like to make changes to.

You will then be redirected to the associated Edit User page.

To add or remove this User to a project, select or unselect the Project's checkbox, then select Save changes.

To change this User's role, select a role from the dropdown menu, then select Save changes.

Note:
Only Project Managers are allowed to edit users.
